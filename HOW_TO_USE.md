# How to Use

1. Install requirements: `pip install -r requirements.txt`
2. Place `lyrics_raw.csv` (with a `raw_lyrics` column) in the project folder.
3. Open the notebook: `jupyter notebook RhymeLM_v2.ipynb`
4. Run all cells top‑to‑bottom to train the model.
5. Generate a verse with: `print(generate_16(model))`
